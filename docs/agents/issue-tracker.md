# Issue tracker: GitHub

Task issues live in GitHub Issues for `dfn-au/dfn-web`.

PRDs live in git under `docs/prds/specs`. Use GitHub Issues for vertical implementation slices and focused decision/deep-dive work created from those PRDs.

Use `gh` from this repo.

- Create: `gh issue create`
- Read: `gh issue view <number> --comments`
- List: `gh issue list`
- Comment: `gh issue comment <number>`
- Label: `gh issue edit <number> --add-label "..." --remove-label "..."`
- Close: `gh issue close <number>`
